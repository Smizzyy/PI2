Übung 1: Routen und CORS mit vert.x

in INF1006 Praktische Informatik 2 von Prof. Dr. Martin Weigel

Willkommen zur ersten Übung von Praktische Informatik 2. Die Übungsblätter können Sie entweder in einem Browser ansehen oder das HTML innerhalb von einem Editor lesen.

Aufgabe 1: PI · 2 = Piepi

Bevor Sie diese Übung starten können müssen Sie die IntelliJ IDE von JetBrains installieren und sich ein vert.x Projekt anlegen. Folgen Sie dafür der Anleitung im Slideset 01-vertx.pdf.

Fügen Sie dem Projekte eine GET-Route hinzu, welche als Rückgabe den Statuscode 200 sendet. Die Webseite soll das Piepi Pokemon und den Text Hello Pi2! anzeigen.

Aufgabe 2: Poke-Team Editor

Erinnern Sie sich noch an den Poke-Team Editor aus der PI1-Vorlesung zu Webservern? Dieser Server erlaubte das Anlegen eines Pokemon Teams. Die Daten wurden auf dem Express-Server gespeichert und bliebe so auch beim Schließen des Browsers erhalten.

Ihre Aufgabe ist es, einen äquivalenten vert.x Server zu schreiben. Der Server soll die gleichen Routen anbieten wir der Express-Server und den gleichen Funktionsumfang besitzen. Gehen Sie wie folgt vor:

  1. Erstellen Sie die notwendigen GET-Routen. Setzen Sie den Statuscode der Rückgaben auf 200 und prüfen Sie, ob die Routen erreichbar sind.
  2. Kopieren Sie den notwendigen HTML-Code der Webseiten aus dem Express-Projekt, welcher sich in diesem Übungsordner befindet und passen Sie diesen an.
  3. Erstellen Sie die POST-Route und legen Sie einen Array mit Pokemon an, dem Sie die gesendeten Pokemon abspeichern.
  4. Ermöglichen Sie einen Cross-Origin Zugriff für die API-Route.
